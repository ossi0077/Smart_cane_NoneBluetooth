# 스마트 지팡이(bluetooth 제거 모델)
- 진행기간 : 2022/11/07 ~ 2022/11/14
## H/W
- Arudino nano
- 조도 센서
- 저항(1K)
- 초음파 센서(HC-SR04)
- 능동 부저
- NeoPixel 바
- 버튼 1개

## 기능
- 주변이 어두울 시 NeoPixel 점등
- 버튼 입력시 알람 소리 발생
- 초음파센서로 물체 감지
  - 초음파 센서 사양(2cm ~ 400cm 거리의 물체 감지)으로 10cm ~ 350cm 의 구간만 구별 가능하게 하였으며 350 초과시 350, 10 미만 시 10으로 측정되도록 지정
  - 알람(진동)단계별 기능
    - 단위 [cm], 단계별로 진동 세기가 달라짐
    1단계 100 ~ 80
    2단계 80 ~ 60
    3단계 60 ~ 40
    4단계 40 ~ 20
    5단계 0 ~ 20
